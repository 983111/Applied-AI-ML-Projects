Abstract
This lab investigates the empirical impact of prompt design strategies on large language model output quality. Eight controlled experiments isolate individual variables — few-shot examples, instruction specificity, chain-of-thought scaffolding, persona assignment, output format constraints, system context richness, instruction polarity, and task decomposition — using K2-Think-v2 as the target model. Each experiment pairs a baseline prompt against a hypothesis-driven enhancement, with outputs evaluated on a structured four-axis rubric (accuracy, clarity, format adherence, task relevance) scored 1–10 by an independent model judge. Methodology
Model
MBZUAI-IFM/K2-Think-v2 via api.k2think.ai — a reasoning-capable open model, temperature 0.8
Evaluation
Automated scoring via a separate judge call: 4-axis rubric (accuracy, clarity, format, relevance), each scored 1–10
Design
Controlled A/B pairs — single variable changed per experiment. Prompt A is always the baseline condition.
Metrics
Composite quality score, response latency (ms), word count, and narrative qualitative analysis per experiment
